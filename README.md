# cwer-plugin v1.0.0

宠物养成插件 —— 基于 Yunzai 框架的1对1宠物养成系统

领养群友做宠物，通过宠爱与调教培养专属关系。缔约后解锁更多互动与玩法。

## 安装

1. 崽根目录拉取：
```
git clone --depth=1 https://github.com/shiomon/cwer-plugin.git ./plugins/cwer-plugin/
```
国内环境：
```
git clone --depth=1 https://ghfast.top/https://github.com/shiomon/cwer-plugin.git ./plugins/cwer-plugin/
```

2. 将 cwer-plugin 文件夹放入 `Miao-Yunzai/plugins/` 或 `Yunzai/plugins/` 目录下，重启即可。

## 功能概览

在群聊中领养宠物，通过宠爱与调教培养亲密度，缔约后解锁更深层的互动。包含状态管理、服装系统、房子系统、成就系统等完整养成玩法。

## 关系系统

- **领养**：70%概率成功，失败有嘲讽。可被抢夺
- **逃避**：领养阶段宠物有概率逃避互动（服从0~99: 40%，100~199: 20%，200~299: 5%），缔约后归零
- **缔约**：亲密≥299，主人发起需宠物30秒内同意，宠物发起直接成功。缔约后双方所有其他宠物/主人关系将被清除
- **解除**：领养/缔约均可解除，30分钟冷却

## 核心机制

- **暴击检定**：每次互动随机RNG(1-100)，≥阈值触发暴击，数值翻倍
- **训练加成**：状态在最优区间内获得加成，装备multiplier额外提升
- **耐久系统**：普通装有耐久，调教指令随机打掉耐久，耐久归零自动卸装
- **金币共享**：主人和宠物共享一个金币池，初始99
- **逃避机制**：领养阶段被动触发，服从越高逃避概率越低，缔约后归零
- **互动冷却**：全局30秒冷却，防止刷屏
- **冷战**：宠物可发起30分钟冷战，期间互动无效
