## 安装

1. 崽根目录拉取：
```
git clone --depth=1 https://github.com/shiomon/cwer-plugin.git ./plugins/cwer-plugin/
```
国内环境：
```
git clone --depth=1 https://ghfast.top/https://github.com/shiomon/cwer-plugin.git ./plugins/cwer-plugin/
```

2. 姐妹插件《猫娘调教》N对1，默认群主

国内环境：
```
git clone --depth=1 https://ghfast.top/https://github.com/shiomon/smer-plugin.git ./plugins/smer-plugin/
```


## 功能概览

在群聊中领养自己的宠物，通过宠爱与调教培养亲密度，缔约后解锁更深层的互动。包含状态管理、服装系统、房子系统、成就系统等完整养成玩法。

## 关系系统

- **领养**：70%概率成功，失败有嘲讽。可被抢夺
- **逃避**：领养阶段宠物有概率逃避互动（服从0~99: 30%，100~199: 15%，200~519: 5%），缔约后归零
- **缔约**：亲密≥99，主人发起需宠物30秒内同意，宠物发起直接成功。缔约后双方所有其他宠物/主人关系将被清除
- **解除**：缔约用`$解除`，领养用`$摆脱`，30分钟冷却

## 成就系统

- **服从类**：初识顺从 → 渐趋顺从 → 死心塌地 → 深度臣服 → 永恒臣服
- **涩气类**：涩气初绽 → 欲念渐起 → 欲念缠身 → 欲壑难填 → 极欲化身
- **亲密类**：依恋之心 → 告白时刻 → 非你不可 → 深情不悔 → 一生一世
- **陪伴类**：黏人的宠物、久经陪伴、长长久久、我爱你、一生一世
- **商店类**：衣不蔽体、碎衣狂魔、初入衣柜、全图鉴、半柜芬芳、迷迭香、神装、足控、善解人衣、衣服是什么、裸体3天、裸体7天
- **魅力类**：小妖精、万众倾倒、绝世魅影
- **特殊类**：M显现、濒临崩溃、麻木、一触即飞、奄奄一息、阎王不收、饥肠辘辘、我饿死也不、你要撑死我、灰姑娘、我有洁癖

## 核心机制

- **暴击检定**：每次互动随机RNG(1-100)，≥阈值触发暴击，数值翻倍
- **训练加成**：状态在最优区间内获得加成，装备multiplier额外提升
- **耐久系统**：普通装有耐久，调教指令随机打掉耐久，耐久归零自动卸装
- **金币共享**：主人和宠物共享一个金币池，初始99
- **逃避机制**：领养阶段被动触发，服从越高逃避概率越低，缔约后归零
- **互动冷却**：全局30秒冷却，防止刷屏
- **冷战**：宠物可发起30分钟冷战，期间互动无效
